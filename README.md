### Цель:
Создать робота который будет сначала анализировать случайный лабиринт, а потом за ограниченное, минимальное возможное время его проходить

### Ограничения:
- Ширина: 168 мм (На самом деле сильно меньше, т.к. это расстояние между стенками, а нам нужно ещё лавировать)
- Длина: 168 мм

### Конкурент: **Green Giant 5.19V**
#### Характеристики
Размеры: 75×105×37 мм
Вес: 118 г с батареей
#### Компоненты и механика
| Компонент      | Характеристики                                                          |
| -------------- | ----------------------------------------------------------------------- |
| **Колёса**     | 3D-печатные с мини-шинами (D=22 мм, ширина 9 мм), редуктор 60:16 (M0.3) |
| **Моторы**     | **Faulhaber 1717R** (coreless, 6 В) + магнитные энкодеры **IE2-512**    |
| **Вентилятор** | **CL-0820-17** (создаёт прижимную силу 151 г/Вт)                        |
| **Батарея**    | **300 мАч 45C LiPo 2S** (7.4 В)                                         |

#### Электроника
| Подсистема              | Компоненты                                                                                        |
| ----------------------- | ------------------------------------------------------------------------------------------------- |
| **МК**                  | **STM32F405RG** (168 МГц, 192 КБ RAM, 1 МБ Flash)                                                 |
| **Драйверы моторов**    | **UCC27524 + DMHC3025LSDQ** (H-bridge), PWM 21 кГц                                                |
| **Драйвер вентилятора** | **UCC27524 + RF4E070GN**                                                                          |
| **Питание**             | **LMZ21701** (5 В), **TPS73633** (3.3 В), **LMZ10501** (2 В), **TPS61088** boost (11.88 В)        |
| **Сенсоры**             | **6× SFH4545** (ИК-излучатели) + **7× TEFT4300** (приёмники), **MPU-6500** гироскоп (20k сэмпл/с) |
| **Интерфейс**           | LED-дисплей **HCMS-2903**, 18 LED, кнопки, buzzer, **Bluetooth BLE 4.0** (DA14580)                |
