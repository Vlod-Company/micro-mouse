### Цель:
Создать робота который будет сначала анализировать случайный лабиринт, а потом за ограниченное, минимальное возможное время его проходить

### Ограничения:
- Ширина: 168 мм (На самом деле сильно меньше, т.к. это расстояние между стенками, а нам нужно ещё лавировать)
- Длина: 168 мм

#### Характеристики
Размеры: 75×105×37 мм
